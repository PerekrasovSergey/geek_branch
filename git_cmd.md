# Подсказки по командной строке

Команда смены директории:
```sh
cd c:\folder_name
```

Команда отображения текущей дириктории Linux, MacOs:
```sh
pwd
```

Листинг следующей директории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```
Удаление файла Win:
```sh
del <filename>
```
Linux, MacOs:
```sh
rm <filename>
```
